🔹 What Are SCDs?  
They track changes in dimension attributes (like customer address or status) that occur slowly rather than frequently.

## 🕰️ Slowly Changing Dimensions (SCD)

### Key Insights
- **Type 1:** Overwrite old data, no history.  
- **Type 2:** Add new rows with effective dates.  
- **Type 3:** Add columns for partial history.  

**Use Case:**  
- Type 1 → Quick updates.  
- Type 2 → Full history tracking.  
- Type 3 → Simple, limited change tracking.
