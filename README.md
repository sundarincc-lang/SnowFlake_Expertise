# Snowflake 90-Day Curriculum

This repo documents my journey from beginner to advanced Snowflake skills in 90 days.

## Structure
- Weeks 1–2: Beginner Foundations
- Weeks 3–4: Data Loading & Warehouses
- Weeks 5–6: Intermediate Data Engineering
- Weeks 7–8: Performance & Optimization
- Weeks 9–10: Advanced Features
- Weeks 11–12: Enterprise & Capstone

## Goals
- Build 3–4 real projects (ETL, optimization, real-time analytics, enterprise pipeline).
- Prepare for SnowPro Core Certification.
- Maintain a portfolio of Snowflake work.
